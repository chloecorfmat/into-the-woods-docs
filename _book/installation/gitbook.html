GitBook

GitBook

GitBook est utilisé pour gérer la documentation, tant d'un point de vue technique que fonctionnel, du projet.

Dans le repository dédié, le dossier _book contient le site statique généré à partir des fichiers markdown.

Commandes utiles

Pour installer gitbook :

npm install gitbook-cli -g

Pour écouter les modifications :

gitbook serve

Pour générer le site statique de la documentation :

gitbook build

Les autres commandes sont disponibles sur la documentation.
